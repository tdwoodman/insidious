##### Installation:

1. Insert low in .emacs: (load "~/.emacs.d/insidious-theme.el")

---

##### Config:

If you don't want Insidious to highlight lines for you, remove this from the theme file:
```
;; set line highlight
(global-hl-line-mode 1)
(set-face-background 'hl-line "gray3")
```
